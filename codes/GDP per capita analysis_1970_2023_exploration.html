Note¶

I think targeting annual GDP growth is more appropriate, but it was hard to get a high-performing regressor model because of the nature of the data series. My guess for the GDP growth prediction is that other types of models like ARIMA, SARIMA, or VAR might be more appropriate (which I'm not super confident with this type of analysis yet).

I conducted an analysis to predict GDP per capita instead, but I’m not confident because of the mixed time series data. Also, I'm still unsure whether our goal is to stick with cross-sectional analysis (in fixed time points) or time series analysis. However, I hope to give us somewhat useful insights from this note.

Createing Lagged features (lag features by 1 year for each country)¶

This captures how past values of features (a year ago) affect current GDP per capita.

In [10]:
def create_lagged_features(df, group_col, time_col, features, lags=[1]):
    df = df.copy()
    df = df.sort_values(by=[group_col, time_col])
    for feature in features:
        for lag in lags:
            df[f"{feature}_lag{lag}"] = df.groupby(group_col)[feature].shift(lag)
            df = df.drop(columns=[feature])
    return df
